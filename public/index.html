    🔌 Configurar Multijugador

    Conecta el juego a Firebase para jugar con tu equipo en tiempo real desde múltiples dispositivos.

    Pega el objeto "firebaseConfig" de tu proyecto de Firebase. Debe incluir "databaseURL".

    💡 ¿Cómo configurar tu Firebase gratis?

    1. Crea un proyecto en console.firebase.google.com.
    2. Agrega una aplicación web y copia el objeto firebaseConfig.
    3. Activa Realtime Database en la consola de Firebase.
    4. En la pestaña Reglas de la base de datos, cámbialas para permitir lectura y escritura temporales o permanentes: 
      {
  "rules": {
    ".read": true,
    ".write": true
  }
}
